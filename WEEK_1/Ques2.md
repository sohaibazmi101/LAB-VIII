## Core Technologies Used in Web Engineering

Web Engineering relies on several core technologies that work together to create functional, interactive, and dynamic web applications. The main technologies include HTML, CSS, JavaScript, and PHP.

---

### 1. HTML (HyperText Markup Language)
HTML is the foundational technology of the web. It is used to structure the content of a web page, such as headings, paragraphs, images, links, tables, and forms. HTML defines the skeleton of a webpage and tells the browser what content to display.

**Role:**
- Creates the basic structure of web pages  
- Organizes content using tags and elements  
- Forms the backbone of any website  

---

### 2. CSS (Cascading Style Sheets)
CSS is used to style and design web pages. It controls the appearance of HTML elements, including layout, colors, fonts, spacing, and responsiveness. CSS helps make web pages visually appealing and user-friendly.

**Role:**
- Enhances the visual presentation of web pages  
- Controls layout and responsiveness for different screen sizes  
- Separates content from design  

---

### 3. JavaScript
JavaScript is a client-side scripting language that adds interactivity and dynamic behavior to web pages. It allows web pages to respond to user actions such as clicks, form submissions, and mouse movements.

**Role:**
- Adds interactivity and dynamic content  
- Validates user input on the client side  
- Enables features like animations, pop-ups, and real-time updates  

---

### 4. PHP (Hypertext Preprocessor)
PHP is a server-side scripting language used to create dynamic web applications. It runs on the server and is commonly used to interact with databases, process form data, manage sessions, and generate dynamic web content.

**Role:**
- Handles server-side logic  
- Connects web applications to databases  
- Processes forms and manages user authentication  

---

### Conclusion
Together, HTML, CSS, JavaScript, and PHP form the core technology stack of web engineering. HTML structures the content, CSS styles it, JavaScript adds interactivity, and PHP handles server-side processing, enabling the development of complete and dynamic web applications.
