## Week 2 - Question 3  
## Major Tools Used in Data Science and Their Purpose

Data Science uses a combination of programming languages, libraries, and platforms to collect, process, analyze, and visualize data efficiently.

---

## 1. Programming Languages

### Python
Python is the most widely used programming language in data science due to its simplicity and rich ecosystem of libraries.

**Purpose:**
- Data analysis and manipulation  
- Machine learning and deep learning  
- Data visualization  

### R
R is a programming language mainly used for statistical computing and data analysis.

**Purpose:**
- Statistical analysis  
- Data modeling  
- Data visualization  

---

## 2. Libraries and Frameworks

### NumPy
A Python library used for numerical computations.

**Purpose:**
- Handling arrays and matrices  
- Performing mathematical operations  

### Pandas
A Python library used for data manipulation and analysis.

**Purpose:**
- Data cleaning and preprocessing  
- Handling structured data  

### Matplotlib / Seaborn
Libraries used for data visualization.

**Purpose:**
- Creating graphs and charts  
- Visualizing data patterns and trends  

### Scikit-learn
A machine learning library in Python.

**Purpose:**
- Building and evaluating machine learning models  
- Data classification, regression, and clustering  

---

## 3. Platforms and Tools

### Jupyter Notebook
An interactive environment for writing and executing code.

**Purpose:**
- Data exploration and analysis  
- Combining code, visualizations, and documentation  

### Google Colab
A cloud-based platform for data science and machine learning.

**Purpose:**
- Running Python code online  
- Free access to GPUs and TPUs  

### Tableau / Power BI
Data visualization and business intelligence tools.

**Purpose:**
- Creating interactive dashboards  
- Business data analysis  

---

## Conclusion
These tools help data scientists efficiently analyze data, build predictive models, and communicate insights, making them essential for modern data-driven applications.
